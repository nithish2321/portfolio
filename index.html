Arithmetic Gamified Learning Platform

A full-stack web application with a futuristic, gamified interface for mastering speed math, featuring an AI companion.

The Challenge

Standard methods for practicing math, like flashcards, can be unengaging. The goal was to create an immersive and motivating learning experience to help users master speed mathematics.

My Process & Approach

I built this platform with a React front end and a Node.js/Express back end, using MongoDB to track user statistics. The Solution

The Speed Math Interface offers a "Jarvis-style" dark-themed UI with multiple game modes and quiz protocols. It includes a dedicated learning hub, comprehensive performance tracking, and an intelligent review system to help users focus on improvement.

Explore Ease - Tour Package Booking Platform

Designed a MERN stack web app with secure user login, profile management, and tour booking via OTP. Users can search packages through an interactive map and manage bookings. Admin dashboard enables package CRUD operations and manages user bookings using a LIFO stack. Added a Python-based NLP chatbot to handle queries and fetch data from MongoDB.

The Challenge

The existing tour booking landscape often lacks comprehensive features, secure user management, and intuitive search functionalities. The challenge was to create a unified platform that addresses these pain points, offering a seamless and engaging experience for both users and administrators.

My Process & Approach

I developed "Explore Ease" using a full MERN stack development cycle. I designed the database schema in MongoDB, built robust RESTful APIs with Node.js and Express.js, and created a dynamic front end with React.js. A key innovation was the integration of a Python-based NLP chatbot to provide real-time assistance.

The Solution

"Explore Ease" delivers a seamless tour booking experience with secure login, an interactive map for package searching, and an OTP-based booking system. The admin dashboard provides powerful tools for managing tour packages and bookings, while the integrated NLP chatbot enhances user support.

Technologies Used

MongoDB Express.js React.js Node.js Python (Flask) NLP OTP Verification

Entrance Ease

Built a collaborative MERN stack platform connecting test centers, colleges, and candidates. Enabled test centers to manage seat availability, and colleges to book centers, upload student data (CSV), and assign slots. Automated hall ticket generation with QR codes and personalized routes, delivered via email. Students can securely access their hall tickets using DOB verification. Backend handles PDF generation and MongoDB-based seat tracking.

The Challenge

Managing entrance exams involves complex coordination between test centers, colleges, and candidates. Manual processes are prone to errors and lack a transparency. The challenge was to create an automated, secure, and collaborative platform to streamline the entire examination process.

My Process & Approach

I developed "Entrance Ease" using the MERN stack. I designed a robust MongoDB database, built a secure Node.js and Express.js back end, and created intuitive React.js front-end interfaces for all stakeholders. Automation was a key focus, particularly for generating personalized hall tickets.

The Solution

"Entrance Ease" is a comprehensive platform that simplifies entrance examination management. It allows test centers to update seat availability, colleges to efficiently book centers and manage student allocations, and students to securely access their hall tickets. The automated hall ticket generation with QR codes and route details significantly reduces manual effort.

MongoDB Express.js React.js Node.js PDF Generation QR Code Email Automation

Campus Garden Monitoring System - Smart Waste Detection

Ongoing research project aimed at publication, using YOLOv11 for detecting upright bins, tilted bins, and trash in campus environments. Features an Android client that uploads images to a Flask server, where a socket-based listener processes data and stores results in MongoDB. A Node.js + React dashboard provides real-time visual monitoring.

The Challenge

Effective waste management in large campus environments is a significant challenge. The goal was to develop an intelligent, automated system to detect waste anomalies and streamline campus clean-up efforts.

My Process & Approach

This research project leverages YOLOv11 for waste detection. The system's architecture includes an Android client for image capture, a Flask server for real-time data processing, and a MongoDB database. A Node.js and React dashboard provides real-time visual monitoring.

The Solution

The Campus Garden Monitoring System offers an automated and proactive solution for waste management. The YOLOv11 model provides highly accurate detection, the Android client allows for flexible image capture, and the real-time dashboard gives administrators a clear overview of waste conditions.

Technologies Used

YOLOv11 Python (Flask) MongoDB Node.js React.js Android Development Computer Vision
